# Employee Management Module (MERN Stack)

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- JWT Authentication (Login / Logout)
- Protected Employee APIs
- Create Employee with Image Upload
- Search & Filter Employees
- Responsive Dashboard UI

## How to Run
###Frontend 

bash
cd client
npm start

### Backend
```bash
cd server
npm install
npx nodemon index.js
